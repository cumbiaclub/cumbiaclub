Cómo funciona
1. La PC crea un hotspot WiFi propio
2. Esta app se conecta a 192.168.137.1:3001
3. El servidor Node.js envía el tema actual en tiempo real
4. Al cambiar de tema en Ableton, el teleprompter carga la letra y resetea el scroll

1. Instalá Node.js en la PC
2. Copiá el script y guardalo como teleprompter-server.js
3. Abrí una terminal y ejecutá: node teleprompter-server.js
4. Dejalo corriendo durante el show
